# Новостной портал — Техническое задание

## Общее описание

API для новостного портала с функционалом только на чтение (вывод данных). Без комментариев, админки, изображений и
видео.

---

## Функциональные требования

### Основные функции

| Функция                     | Пример                                        |
|-----------------------------|-----------------------------------------------|
| Показывать список новостей  | https://ria.ru/politics/                      |
| Показать конкретную новость | https://ria.ru/20260227/putin-2077144270.html |

### Категории и теги

| Сущность             | Ограничения                                           |
|----------------------|-------------------------------------------------------|
| Категории            | Ограниченный список (в мире, политика, погода, ...)   |
| Теги                 | До 100 ключевых слов (Самолеты, Toyota, Игрушки, США) |
| Теги на новость      | До 10 в среднем                                       |
| Категория на новость | Одна (1:N)                                            |

### Формат и фильтрация

| Параметр      | Значение                    |
|---------------|-----------------------------|
| Формат выдачи | HTML                        |
| Фильтрация    | По периоду, категории, тегу |
| Пагинация     | Есть                        |

## Бизнес-сущности

### Новость (News)

| Поле        | Тип          | Обязательное | Описание                     |
|-------------|--------------|--------------|------------------------------|
| title       | text         | *            | Название                     |
| preamble    | text         | *            | Преамбула (краткое описание) |
| content     | text         |              | Содержание (HTML)            |
| category    | Category     | *            | Категория                    |
| tags        | Tag[]        |              | Теги (до 10 шт)              |
| author      | varchar(255) | *            | Автор                        |
| createdAt   | datetime     | *            | Дата создания                |
| publishedAt | datetime     |              | Дата публикации              |
| status      | Status       | *            | Статус                       |

### Статусы новости

| Значение    | Описание       |
|-------------|----------------|
| published   | Опубликован    |
| unpublished | Не опубликован |
| deleted     | Удален         |

### Категория (Category)

| Поле      | Тип    | Обязательное | Описание            |
|-----------|--------|--------------|---------------------|
| name      | text   | *            | Название на русском |
| sortOrder | int    | *            | Порядок отображения |
| status    | Status | *            | Статус              |

### Тег (Tag)

| Поле   | Тип    | Обязательное | Описание |
|--------|--------|--------------|----------|
| name   | text   | *            | Название |
| status | Status | *            | Статус   |

## HTTP JSON API

### Endpoints

#### GET /v1/listNews

Список новостей с фильтрацией и пагинацией.

**Query параметры:**

| Параметр | Тип      | Описание                        |
|----------|----------|---------------------------------|
| category | int      | ID категории                    |
| tag      | int      | ID тега                         |
| from     | datetime | Начало периода                  |
| to       | datetime | Конец периода                   |
| page     | int      | Номер страницы (по умолчанию 1) |
| limit    | int      | Количество на страницу          |

**Ответ:** Краткий формат (без content, с полными категориями и тегами)

#### GET /v1/countNews

Количество новостей по фильтру.

**Query параметры:**

| Параметр | Тип      | Описание       |
|----------|----------|----------------|
| category | int      | ID категории   |
| tag      | int      | ID тега        |
| from     | datetime | Начало периода |
| to       | datetime | Конец периода  |

**Ответ:** `{ "count": number }`

#### GET /v1/getNews

Получить конкретную новость.

**Query параметры:**

| Параметр | Тип | Описание   |
|----------|-----|------------|
| id       | int | ID новости |

**Ответ:** Полный формат (с content)

#### GET /v1/getCategories

Получить список доступных категорий.

**Ответ:** Массив категорий

#### GET /v1/getTags

Получить список тегов.

**Ответ:** Массив тегов

---

## Форматы ответов

### Краткий (список)

- Список новостей
- Полные теги
- Полные категории
- Полные авторы
- **Без** содержания (content)

### Полный (конкретная новость)

- Все поля новости включая content
